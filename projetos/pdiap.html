Sobre o projeto

O PDIAP é um projeto de pesquisa e desenvolvimento do IFSUL Campus Venâncio Aires, criado para ser uma plataforma digital para administração de projetos de feiras científicas de forma dinâmica e automatizada.

Trabalho no PDIAP desde 2018, hoje atuando como voluntário. O projeto é desenvolvido utilizando a MEAN Stack, um conjunto de tecnologias voltado para aplicações WEB.

Este projeto contou com a colaboração de diversos colegas ao longo de sua elaboração.
