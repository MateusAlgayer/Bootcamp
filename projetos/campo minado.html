Sobre o projeto

Este projeto foi desenvolvido para a cadeira de linguagem de programação II do curso superior em Análise e Desenvolvimento de Sistemas. O objetivo era criar o jogo “Campo Minado” utilizando apenas o CMD para exibir os campos.

Habilidades desenvolvidas

Criação e manipulação de listas
Criação e manipulação de objetos
Uso de Exceptions e tratamentos de erro
Validações de entrada do usuário
Herança e polimorfismo

Como testar?

Para jogar, basta baixar o projeto e executá-lo no NetBeans ou outra IDE de sua escolha.

Você também pode utilizar o arquivo .jar disponível no GitHub, mas será necessário desabilitar as cores do jogo. :(
